CHAPSim_post
------------
This is the documentation for CHAPSim_post, a postprocessing Python code for the CHAPSim DNS solver. This contains guides for using the modules. There is also some documentation containing function headers but these are limited. It should be noted that this code is updated and maintained on an ad hoc basis, bugs and issues can be put on the code's GitHub page `here <https://github.com/mattfalcone1997/CHAPSim_post>`_.
